Playlist to text converter

Reads .M3U, .M3U8, .PLS and .ASX








Example:


		

	

	
Warning
Some songs with missing title.
Filenames were used instead.

	
	

	
Filenames will act as titles if no metadata is found.
Underscores in filenames will be converted to spaces.

	
Download for offline use: https://github.com/fluoroom/playlist-2-text/archive/master.zip

	
GitHub repository: https://github.com/fluoroom/playlist-2-text/
